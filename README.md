#  Fake News Detection (NLP Project)

##  Overview
A Natural Language Processing (NLP) project that classifies news articles as **fake or real** using machine learning techniques.

---

##  Approach
- Text preprocessing (cleaning, stopwords removal, lemmatization)
- Feature extraction using TF-IDF
- Classification using Logistic Regression

---

##  Results
- Accuracy: 90.93%
- Evaluated using precision, recall, F1-score, and confusion matrix

---

##  Tools
- Python
- NLTK
- Scikit-learn
- Gensim
- Matplotlib & Seaborn

---

##  Dataset
Kaggle. (2017). Fake and Real News Dataset. Retrieved from
https://www.kaggle.com/datasets/jillanisofttech/fake-or-real-news?resource=download
---

##  Goal
To demonstrate how NLP techniques can be used for fake news detection and text classification.
